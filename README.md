# Getting Started

## Paso 1
Crear archivo .env en la carpeta raiz

Dentro del archivo .env agregar tu Api key de https://newsapi.org/

```bash
API_KEY=api_key
```

## Paso 2
```bash
npm i
```

## Paso 3
```bash
npx pod-install ios
```

## Ejecutar android
```bash
npm run android
```

## Ejecutar ios
```bash
npm run ios
```
